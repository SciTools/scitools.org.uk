Interpolation and re-gridding routines.

See also: NumPy, and SciPy.

Deprecated since version 1.10: The module iris.analysis.interpolate is deprecated. Please use iris.cube.regrid() or iris.cube.interpolate() with the appropriate regridding and interpolation schemes from iris.analysis instead.

The actual content of this module is all taken from ‘iris.analysis._interpolate_backdoor’. The only difference is that this module also emits a deprecation warning when it is imported.
