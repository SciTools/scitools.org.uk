iris.experimental.concatenate¶

Automatic concatenation of multiple cubes over one or more existing dimensions.

Warning

Currently, the concatenate() routine will load the data payload of all cubes passed to it.

This restriction will be relaxed in a future release.

In this module:

  • concatenate
iris.experimental.concatenate.concatenate(cubes)¶

Concatenate the provided cubes over common existing dimensions.

Args:

  • cubes:

    An iterable containing one or more iris.cube.Cube instances to be concatenated together.

Returns:
A iris.cube.CubeList of concatenated iris.cube.Cube instances.

Warning

This routine will load your data payload!
