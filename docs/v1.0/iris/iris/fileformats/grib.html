iris.fileformats.grib¶

Conversion of cubes to/from GRIB.

See also: ECMWF GRIB API.

iris.fileformats.grib.add_load_rules(filename)¶

Registers a rules file for use during the GRIB load process.

Registered files are processed after the standard rules, and in the order they were registered.

iris.fileformats.grib.grib_generator(filename)¶

Returns a generator of GribWrapper fields from the given filename.

iris.fileformats.grib.load_cubes(filenames, callback=None)¶

Returns a generator of cubes from the given list of filenames.

iris.fileformats.grib.reset_load_rules()¶

Resets the GRIB load process to use only the standard conversion rules.

iris.fileformats.grib.save_grib2(cube, target, append=False, **kwargs)¶

Save a cube to a GRIB2 file.

Args:

  • cube - A iris.cube.Cube, iris.cube.CubeList or list of cubes.
  • target - A filename or open file handle.

Kwargs:

  • append - Whether to start a new file afresh or add the cube(s) to the end of the file.

    Only applicable when target is a filename, not a file handle. Default is False.

See also iris.io.save().

Contains a pygrib object plus some extra keys of our own.
